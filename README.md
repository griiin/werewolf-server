An adaption of the namesake board game - server part

1- Install node packages
  run:
  $> npm install

2- Fix grunt jasmine node
  run:
  $> chmod +x jasmine_node_patch.sh && ./jasmine_node_patch.sh

3- Launch a mongodb

4- Run server
  run:
  $> node app.js
